FoxBot Getting Started

 

Getting Started

 

In order to play Team Fortress Classic with the bots, it is important to be familiar with a few basic commands. All the commands here should be typed in the console while in game. The console can be accessed in game by pressing the tilde key.

 

Enabling the console

The console is not usually accessible by default. You have to allow access to it and here's how:
You need to edit the shortcut that starts up Half-Life or TFC.
Add this to the end of the shortcut:
-console

 

Accessing the Console within the game

 

1) Run Team Fortress Classic from Steam or the shortcut on your desktop.

2) Once in the game, choose Create Server.

3) Change the maximum number of players to you want to have allowed (using the game tab).

This number represents the maximum number of bots and humans for both teams.

4) Choose the map that you want to play. (make sure it is a map that has waypoints for it.)

5) Start game – Once in the game hit the tilde key (~) to hide and show the console.

The console is where you enter the bot commands as shown in the commands section.
