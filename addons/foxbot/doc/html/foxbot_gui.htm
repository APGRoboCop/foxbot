FoxBot GUI

 

Mini GUI by GoaT_RopeR, now available with FoxBot

It is a condensed version of the FoxBot GUI which will be released in the future.

Its purpose is to easily allow users to change many of the settings of the bot without editing cfg files.

 

 

Main Page - Left Window

1  Clicking BLUE links will take you to that mods homepage

2  GREEN links are shown when something is currently loaded. Clicking on green links will UNLOAD that mod.

3  RED Links are shown when something is unloaded. Clicking on red links will LOAD that mod.

4  Mod Version numbers are shown in PURPLE text.

 

Main Page - Right Window

1  Shows the status of any mods that are currently running.

2  If the mods are in memory and running (hlds for example) this box will show them as running.

 

 

Main Page - Info Window

1  Info Window contains current versions of installed programs, and warns the user if they have an older version of the program with (Old version) as shown above.

2  In addition to telling you (Old version) a mouseover on the version number will show you the latest known version.

3  Clicking Yes Next to "<modname> Loaded" will UNLOAD that mod. Clicking No will LOAD that mod.

4  When loading a mod, the GUI will show an error if you do not have the mod. (Metamod, Admin-Mod, and Neo-TF, must all be downloaded seperately)

5  When loading mods, the GUI will load whatever mods you choose using the most stable and working version that we have tested.

6  GUI will give an error if mods are loaded incorrectly due to manual configurations.

7  Clicking the name of the mod will take your browser to the mods homepage.

8  GUI will notify users if any of the mods are missing needed files.

 

 

Configure FoxBot Page

1  Edit the default skill level from 1-5 (1=best,5=worst) with a slider.

2  Edit the pause time before bots begin to join with a slider.

3  Edit the min_bots value to determine the minimum # of bots you want in your game.

4  Edit the max_bots value to determine the maximum # of bots you want in your game.

5  Toggle on/off - bot_chat, bot_xmas, bot_balance, bot_team_balance

6  Load and edit the foxbot_chat.txt file which the bots use when bot_chat is enabled.

7  Load and edit the foxbot_names.txt which lists the available names for a bot to choose from when joining.
